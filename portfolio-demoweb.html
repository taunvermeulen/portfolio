A Demo Website ForAn Interior App

Interiored

This is a demo website for an app that is made to connect clients with interior designers. The idea of the app is based on tinder.

With this demo website, I just wanted to figure out how to use CSS to make a website look good. I also used this experience to familiarise myself with Bootstrap.

This was a fun way to see what I had learnt from both the coding side of just HTML and CSS, while also testing my creativity to make a good-looking website that represents the product. I also learned that even though bootstrap helps guide one's CSS, you are not forced into a certain style when using the framework.
