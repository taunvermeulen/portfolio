Formula OneQuiz App

F1 Quiz App

A Quiz App that has a set number of questions on Formula One general knowledge. The app has a countdown to give off some pressure! However, one can always retake the quiz with a push of a button. The quiz also shows the user his or her score afterward.

When making the quiz, I wanted to focus on making a web app that has a continuous loop so one can retry the quiz without needing to press the refresh button. I wanted to implement a scoring system so users can see how well they know some Formula One general knowledge and compare it with each other.

I enjoyed this project! It showed me that there is always a way forward when getting stuck. This project helped me become more comfortable when working with JavaScript.
