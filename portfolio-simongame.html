An Interactive Memory Game

Simon's Game

Simon's game is a fun interactive memory game, that is based on an old board game. The main idea of the game is to continuously remember consecutive colours and sounds to get a higher score than your friends and family.

When making this game, there were a few things that I intended to learn from the experience. I wanted to learn how to effectively use event listeners to make the game interactive, make use of variables to keep a score, use a loop to keep the game going and restart the game when one doesn't succeed, better my commenting skills, and finally make use of jQuery to shorten my code.

I enjoyed this project as it felt like making a real game that becomes more and more challenging. I saw how a fraction of an old framework can save so much time and effort.
